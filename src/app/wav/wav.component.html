WAV Functions

WAV is an alternative to the intrinsics API for C/C++. It was designed to be safer and easier to use than the intrinsics API while providing the same performance.

Note that not all WAV functions are currently listed here; overloaded aliases (for example, wav_add instead of wav_i32x4_add, wav_i64x2_add, wav_f32x4_add, etc.) are not shown, nor are the various conversion functions. For more documentation of WAV, please see the project page.
